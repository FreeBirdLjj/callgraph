#+TITLE: callgraph

This repository currently provides 1 command =callgraph= (=./cmd/callgraph=) to analyze Go codes with call graphs.

* Command: =callgraph=

The command =callgraph= (=./cmd/callgraph=) provides some subcommands.

** Subcommand: =draw=

Example command:

#+BEGIN_SRC bash
$ callgraph draw unicode
#+END_SRC
